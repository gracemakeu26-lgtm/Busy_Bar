BUSY Bar est un disposoitif multi-outil de productivité avec un écran LED pixelisé. Minuteur de concentration avec fonction de blocage des distractions sur votre téléphone et PC. Entièrement personnalisable, open-source et prêt pour la maison connectée.

Outil de productivité

Blocage des distractions, message BUSY personnalisé, minuteur de concentration, synchronisation multiplateforme

Application et intégration

bibliothèque d'applications, connexion à des logiciels tiers, intégrations avec les événements de calendrier et les appels

Support maison connectée

Connectez-vous à Google Home et Apple Home via le protocole Matter

Pour les développeurs

API HTTP ouvrte, SDK open-source,bibliothèques Python / GO / JavaScript, MQTT, sans verrouillage fournisseur

Montrez que vous êtes occupé

Empêchez les autres d'interrompre votre concentration profonde. L'écran affiche que vous êtes occupé et quand vous serez à nouveau disponible

Activation instantanée

Activez et desactivez le statut BUSY manuellement avec un large boutton. Fonctionne entièrement hors ligne, sans logicicel supplementaire requis

Set timer

Set an optional expiration timer BUSY using the scroll wheel t o let other know when you'll be available again.
